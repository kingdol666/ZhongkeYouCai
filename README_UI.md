# CSV 数据批处理工具

这是一个基于 PyQt5 的图形界面工具，用于批处理 CSV 数据文件。

## 功能特点

- 可视化界面操作，简单易用
- 支持自定义数据行范围（起始行号和终止行号）
- 支持自定义模头位置范围（起始位置和终止位置）
- 多线程处理，界面不会卡顿
- 实时显示处理进度和状态
- 自动跳过数据点数量不匹配的文件

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行 UI 界面：

   ```bash
   python run_ui.py
   ```

2. 在界面中设置参数：

   - **输入文件夹**：包含 CSV 文件的文件夹路径<span style="color: red;">【注意，文件夹需要满足有许多子文件夹，每个子文件夹中包含 CSV 文件如：21_51_29_1.CSV】</span>
   - **输出文件夹**：处理后的 CSV 文件保存路径
   - **起始行号**：数据开始的行号（默认为 8）
   - **终止行号**：数据结束的行号（默认为 190）
   - **起始模头位置**：模头起始位置（默认为 2.50）
   - **终止模头位置**：模头终止位置（默认为 14.39）

3. 点击"开始处理"按钮开始处理数据

4. 处理过程中可以点击"停止处理"按钮中断处理

## 输入文件格式

- CSV 文件应包含位置和厚度数据
- 文件名格式为：`时_分_秒_序号.CSV`（例如：`21_51_29_1.CSV`）
- 数据从指定起始行号开始，到指定终止行号结束

## 输出文件格式

- 处理后的文件保存为`merged_子文件夹名.csv`
- 行为时间，列为模头位置
- 每个单元格包含对应时间和位置的膜厚数据

## 注意事项

- 确保输入文件夹中包含子文件夹，每个子文件夹中包含 CSV 文件
- 只有数据点数量与预期完全匹配的文件会被处理
- 处理过程中请勿关闭程序
